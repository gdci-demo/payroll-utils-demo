# Payroll Utils Demo

This repository is a demo project for GitHub Projects, Codespaces, Copilot, Actions, Advanced Security, pull requests, and repository governance.

It provides a simple Python payroll utility package with a deliberately opaque payroll ID validation helper meant for later explanation and refactoring.

## Structure

- `.devcontainer/devcontainer.json` - Python dev container configuration
- `payroll/__init__.py` - package initializer
- `payroll/utils.py` - payroll utility module
- `README.md` - demo repository overview
